# graphing-calc-minimal
A minimal graphing calculator for graphing functions accurately and efficiently

## features
- Scales for both x, y, and both
- Support for a wide variety of functions
- Options to change resolution, thickness of graph

- Fix lines being drawn over asymptotes
- Add support for changing center of graph
